Restructure the Thinkit category system according to the following final business hierarchy.

Requirements:

1. Create all main categories and subcategories exactly as listed below.

2. The Admin Panel must support:
- Adding new categories
- Editing category names
- Adding category cover images
- Adding new subcategories
- Editing subcategories
- Assigning products to both a main category and a subcategory

3. The Customer App must:
- Show only main categories on the Categories page.
- When a customer clicks a main category, show horizontal subcategory tabs.
- Clicking a subcategory should filter products instantly without reloading the page.
- If no products exist, show:
  "No products available in this section."

4. The system must be mobile responsive and optimized for quick-commerce UX.
