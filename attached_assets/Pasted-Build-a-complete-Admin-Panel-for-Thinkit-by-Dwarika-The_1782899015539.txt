Build a complete Admin Panel for Thinkit by Dwarika.

The Admin Panel should be simple, fast, mobile-friendly, desktop-friendly, and optimized for managing a single grocery store (Dwarika Grocery Mart).

Do NOT add unnecessary complexity such as multi-admin support, delivery boy apps, GPS tracking, or analytics dashboards.

The design must follow Thinkit's green and yellow branding and maintain a clean, premium, modern UI.

==================================================
1. ADMIN LOGIN
==================================================

Requirements:

- Separate Admin Login Page
- Username + Password authentication
- Only one admin account is required for v1.0

Default credentials:

Username: admin
Password: admin123

After successful login:
Redirect to Admin Dashboard.

Create a clean dashboard with the following cards:

Add:

Recent Orders Section

Show:
- Customer Name
- Order ID
- Order Value
- Status
- Time

3. NEW ORDER ALARM SYSTEM
==================================================

When a new order arrives:

- Play a loud notification sound continuously.
- Show a popup:

NEW ORDER RECEIVED

Customer Name
Order Amount

Buttons:

Accept Order
View Order

The alarm should stop ONLY when:

Accept Order is pressed.

Do not auto-stop after a few seconds.

8. IN STOCK / OUT OF STOCK TOGGLE
==================================================

Each product should have:

IN STOCK
OUT OF STOCK

One-click toggle.

If a product is marked:

OUT OF STOCK

It should immediately disappear from the customer app and show:

Out of Stock

Customers must not be able to order it.

This is extremely important for Thinkit because inventory is managed manually in the physical store.

==================================================
